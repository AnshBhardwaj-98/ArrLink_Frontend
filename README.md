# ArrLink Website

A modern React website built with Vite, TypeScript, and shadcn-ui.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or bun

### Installation

```sh
npm install
```

### Development

```sh
npm run dev
```

### Build

```sh
npm run build
```

### Preview Production Build

```sh
npm run preview
```

## Available Scripts

- `dev` - Start development server
- `build` - Build for production
- `build:dev` - Build in development mode
- `lint` - Run ESLint
- `preview` - Preview production build
- `test` - Run tests
- `test:watch` - Run tests in watch mode
